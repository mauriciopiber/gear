Teste\FunctionalTester::waitForElementChange

[!***REMOVED*** Method is generated. Documentation taken from corresponding module.

Waits up to $timeout seconds for the given element to change. Element "change" is determined by a callback function which is called repeatedly until the return value evaluates to true. ``` php <?php $I->waitForElementChange('#menu', function(\WebDriverElement $el) { return $el->isDisplayed(); }, 100); ?> ```

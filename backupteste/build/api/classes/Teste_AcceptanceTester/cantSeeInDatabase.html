Teste\AcceptanceTester::cantSeeInDatabase

[!***REMOVED*** Method is generated. Documentation taken from corresponding module.

Effect is opposite to ->seeInDatabase Checks if there is no record with such column values in database. Provide table name and column values. Example: ``` php <?php $I->dontSeeInDatabase('users', array('name' => 'Davert', 'email' => 'davert@mail.com')); ``` Will generate: ``` sql SELECT COUNT(*) FROM `users` WHERE `name` = 'Davert' AND `email` = 'davert@mail.com' ``` Fails if such user was found.

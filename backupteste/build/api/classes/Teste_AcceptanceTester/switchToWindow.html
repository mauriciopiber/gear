Teste\AcceptanceTester::switchToWindow

[!***REMOVED*** Method is generated. Documentation taken from corresponding module.

Switch to another window identified by name. The window can only be identified by name. If the $name parameter is blank, the parent window will be used. Example: ``` html <input type="button" value="Open window" onclick="window.open('http://example.com', 'another_window')"> ``` ``` php <?php $I->click("Open window"); # switch to another window $I->switchToWindow("another_window"); # switch to parent window $I->switchToWindow(); ?> ``` If the window has no name, the only way to access it is via the `executeInSelenium()` method, like so: ``` php <?php $I->executeInSelenium(function (\Webdriver $webdriver) { $handles=$webdriver->getWindowHandles(); $last_window = end($handles); $webdriver->switchTo()->window($last_window); }); ?> ```

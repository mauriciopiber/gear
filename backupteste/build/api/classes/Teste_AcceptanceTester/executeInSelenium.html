Teste\AcceptanceTester::executeInSelenium

[!***REMOVED*** Method is generated. Documentation taken from corresponding module.

Low-level API method. If Codeception commands are not enough, this allows you to use Selenium WebDriver methods directly: ``` php $I->executeInSelenium(function(\WebDriver $webdriver) { $webdriver->get('http://google.com'); }); ``` This runs in the context of the [RemoteWebDriver class***REMOVED***(https://github.com/facebook/php-webdriver/blob/master/lib/remote/RemoteWebDriver.php). Try not to use this command on a regular basis. If Codeception lacks a feature you need, please implement it and submit a patch.
